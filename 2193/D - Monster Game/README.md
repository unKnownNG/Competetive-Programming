<div class="header"><div class="title">D. Monster Game</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You have been gifted a new game called "Elatyh". In the game, you are given $$$n$$$ swords, each with its own strength. In particular, the sword numbered $$$i$$$ has a strength of $$$a_i$$$. The game consists of $$$n$$$ levels, each of which features a monster.</p><p>You start at level $$$1$$$ and progress further. To pass level $$$i$$$ and move on to level $$$i + 1$$$, you need to defeat the monster at level $$$i$$$. To defeat the monster at level $$$i$$$, you need to deal it $$$b_i$$$ sword strikes. The swords in the game are very fragile, so they can only deal <span class="tex-font-style-bf">one</span> strike before breaking. If you complete level $$$n$$$ or run out of swords, you can finish the game and proceed to score calculation.</p><p>Before the game, you are allowed to choose the difficulty level. If you choose difficulty $$$x$$$, swords with a strength less than $$$x$$$ will not affect the monsters. The game score in this case is equal to $$$x$$$ multiplied by the number of levels completed. Your task is to choose the game difficulty in such a way as to maximize the game score.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test consists of several test cases. The first line contains a single integer $$$t$$$ ($$$1\le t\le 10^4$$$) — the number of test cases. The following describes the test cases.</p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1\le n\le 2 \cdot 10^5$$$).</p><p>The second line of each test case contains $$$n$$$ integers $$$a_1, a_2, \dots, a_n$$$ $$$(1\le a_i\le 10^9)$$$.</p><p>The third line of each test case contains $$$n$$$ integers $$$b_1, b_2, \dots, b_n$$$ $$$(1\le b_i\le n)$$$.</p><p>It is guaranteed that the sum of the values of $$$n$$$ across all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single integer — the maximum game score.</p></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div class="input"><div
