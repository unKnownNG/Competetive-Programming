<h3><a href="https://codeforces.com/contest/2176/problem/B" target="_blank" rel="noopener noreferrer">Optimal Shifts</a></h3>

<div class="header"><div class="title">B. Optimal Shifts</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given a binary string $$$s_1s_2 \ldots s_n$$$, containing at least one <span class="tex-font-style-tt">1</span>. You want to obtain a binary string of the same length, consisting only of <span class="tex-font-style-tt">1</span>s. To do this, you can perform the following operation any number of times:</p><p>Choose a number $$$d$$$ ($$$1 \le d \le n$$$) and consider the string $$$t$$$ as a cyclic right shift of the string $$$s$$$ by $$$d$$$, or, more formally, $$$t = s_{n - d + 1} \ldots s_{n}s_{1} \ldots s_{n - d}$$$. After that, for all $$$j$$$ for which $$$t_j = 1$$$, perform $$$s_j := 1$$$. The described operation costs $$$d$$$ coins, where $$$d$$$ is the chosen shift amount.</p><p>Note that the positions $$$j$$$ in the string $$$s$$$, where initially $$$s_j=1$$$, remain equal to $$$1$$$ even if $$$t_j=0$$$.</p><p>You need to determine the minimum number of coins that can be spent so that the string $$$s$$$ consists only of <span class="tex-font-style-tt">1</span>s after all operations.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains the number $$$n$$$ ($$$1 \le n \le 2 \cdot 10^5$$$) — the size of the given binary string.</p><p>The second line of each test case contains a binary string of length $$$n$$$, each element of which is either <span class="tex-font-style-tt">0</span> or <span class="tex-font-style-tt">1</span>.</p><p>It is guaranteed that at least one character in each string is equal to <span class="tex-font-style-tt">1</span>.</p><p>It is guaranteed that the sum of $$$n$$$ across all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output the answer to it — the minimum possible number of coins that you can spend to make all characters in the string equal to <span class="tex-font-style-tt">1</span>.</p></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div
