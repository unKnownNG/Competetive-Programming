## Problem Statement

<div class="header"><div class="title">A. Bigrams</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>A bigram in a string is a pair of adjacent characters. For example, the string <span class="tex-font-style-tt">helloello</span> contains $$$8$$$ bigrams: <span class="tex-font-style-tt">he</span>, <span class="tex-font-style-tt">el</span>, <span class="tex-font-style-tt">ll</span>, <span class="tex-font-style-tt">lo</span>, <span class="tex-font-style-tt">oe</span>, <span class="tex-font-style-tt">el</span>, <span class="tex-font-style-tt">ll</span>, <span class="tex-font-style-tt">lo</span>.</p><p>Monocarp has cards with letters: $$$c_1$$$ cards with the letter <span class="tex-font-style-tt">a</span>, $$$c_2$$$ cards with the letter <span class="tex-font-style-tt">b</span>, ..., $$$c_k$$$ cards with the $$$k$$$-th letter of the Latin alphabet. He wants to make a string from these cards, using each card <span class="tex-font-style-bf">exactly once</span>. The resulting string must contain at least two equal bigrams. The order of characters in each bigram matters; for example, the string <span class="tex-font-style-tt">aba</span> does not have two equal bigrams.</p><p>Determine whether it is possible to make a string that satisfies these requirements.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains one integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test cases.</p><p>Each test case consists of two lines:</p><ul> <li> the first line contains one integer $$$k$$$ ($$$1 \le k \le 10$$$); </li><li> the second line contains $$$k$$$ integers $$$c_1, c_2, \dots, c_k$$$ ($$$1 \le c_i \le 10^8$$$), where $$$c_i$$$ is the number of cards with the $$$i$$$-th letter of the Latin alphabet. </li></ul></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output <span class="tex-font-style-tt">YES</span> if it is possible to construct a string satisfying the condition, or <span class="tex-font-style-tt">NO</span> otherwise.</p><p>Each letter can be output in any case. For example, <span class="tex-font-style-tt">yes</span>, <span class="tex-font-style-tt">Yes</span>, <span class="tex-font-style-tt">yEs</span> will be recognized as a positive answer.</p></div><div
