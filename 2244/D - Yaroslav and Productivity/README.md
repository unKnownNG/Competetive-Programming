His productivity can be negative if he watches short videos, or positive if he works. The <span class="tex-font-style-it">total productivity</span> is defined as the <span class="tex-font-style-bf">sum</span> of all values in the array.</p><p>Sometimes Yaroslav reads motivational posts. He has $$$m$$$ posts, where the $$$j$$$-th post has an impact value $$$b_j$$$. If Yaroslav reads a post with value $$$b_j$$$, then all productivity values from the beginning of the day up to position $$$b_j$$$ change sign, that is, all integers $$$a_1, a_2, a_3, \dots, a_{b_j}$$$ are multiplied by $$$-1$$$.</p><p>For example, let $$$a = [1, -4, 3, -4]$$$, and Yaroslav reads a post with value $$$3$$$. Then the signs of the first three elements are flipped, and the array becomes $$$[-1, 4, -3, -4]$$$. If he then reads a post with value $$$1$$$, the first element changes sign again, and the array becomes $$$[1, 4, -3, -4]$$$.</p><p>What is the <span class="tex-font-style-bf">maximum</span> possible <span class="tex-font-style-it">total productivity</span> Yaroslav can achieve by reading any (possibly none) of the posts?</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains a single integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test cases.</p><p>The first line of each test case contains two integers $$$n$$$ and $$$m$$$ ($$$1 \le m \le n \le 2 \cdot 10^5$$$) — the number of measurements and the number of motivational posts.</p><p>The second line of each test case contains $$$n$$$ integers $$$a_i$$$ ($$$-10^9 \le a_i \le 10^9$$$) — the productivity values.</p><p>The third line of each test case contains $$$m$$$ integers $$$b_i$$$ ($$$1 \le b_i \le n$$$) — the impact values of the posts. It is guaranteed that all $$$b_i$$$ are distinct.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div
