<h3><a href="https://codeforces.com/contest/2194/problem/B" target="_blank" rel="noopener noreferrer">Offshores</a></h3>

<div class="header"><div class="title">B. Offshores</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Mark loves money very much, and he keeps most of it in the bank. He stores his money in $$$n$$$ different banks. In the $$$i$$$-th bank, he has $$$a_i$$$ rubles.</p><p>One day, Mark decided to gather all his money in one bank, so he will be transferring money from one bank account to another. All interbank transfers are arranged the same way: he can transfer only $$$x$$$ rubles in one transfer, and considering all fees, $$$y$$$ rubles will be credited to the other account (since banks want to make a profit, it holds that $$$y \leq x$$$).</p><p>It is possible that Mark will not be able to transfer all his money to one bank, but he wants to find the maximum number of rubles that can end up in any bank.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains three integers $$$n$$$, $$$x$$$, and $$$y$$$ ($$$2 \le n \le 2 \cdot 10^5$$$, $$$1 \le y \le x \le 10^9$$$) — the number of banks, the transfer amount, and the credited amount.</p><p>The second line of each test case contains $$$n$$$ integers $$$a_1, a_2, \ldots, a_n$$$ ($$$1 \le a_i \le 10^9$$$) — the initial amount of rubles in each bank.</p><p>It is guaranteed that the sum of the values of $$$n$$$ across all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single number — the maximum number of rubles that can be obtained in any bank.</p></div><div
