<h3><a href="https://codeforces.com/contest/2225/problem/B" target="_blank" rel="noopener noreferrer">Alternating String</a></h3>

<div class="header"><div class="title">B. Alternating String</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Let us call a string $$$t$$$ <span class="tex-font-style-it">alternating</span> if for every $$$i$$$ from $$$1$$$ to $$$n - 1$$$, the condition $$$t_{i} \neq t_{i + 1}$$$ holds.</p><p>You are given a string $$$s$$$ consisting only of the letters "<span class="tex-font-style-tt">a</span>" and/or "<span class="tex-font-style-tt">b</span>". You may perform the following operation on it <span class="tex-font-style-bf">at most once</span>:</p><ul> <li> choose a substring $$$s_{l}s_{l+1}\ldots s_{r}$$$ consisting of at least one character; </li><li> after choosing the substring, you may invert all letters in it, that is, change every "<span class="tex-font-style-tt">a</span>" to "<span class="tex-font-style-tt">b</span>" and every "<span class="tex-font-style-tt">b</span>" to "<span class="tex-font-style-tt">a</span>" (you are allowed to do it, but you don't have to do it); </li><li> reverse the chosen substring, that is, transform the string $$$s_{1}s_{2}\dots s_{l-1}s_{l}s_{l+1}\ldots s_{r}s_{r+1}\ldots s_{n}$$$ into $$$s_{1}s_{2}\dots s_{l-1}s_{r}s_{r-1}\ldots s_{l}s_{r+1}\ldots s_{n}$$$. </li></ul><p>Note that when performing such an operation, you are not required to do the second step. For example, for the string $$$s =~$$$"<span class="tex-font-style-tt">ababbab</span>", after one operation you can obtain "<span class="tex-font-style-tt">abababa</span>" by choosing the substring $$$s_{5}s_{6}s_{7}$$$ and doing the second step, or "<span class="tex-font-style-tt">bababab</span>" by choosing the substring $$$s_{1}s_{2}s_{3}s_{4}$$$ and not doing the second step. However, you always have to perform the third step of the operation. </p><p>Your task is to determine whether it is possible to obtain any <span class="tex-font-style-it">alternating</span> string from the string $$$s$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>Each test case consists of one line containing one string $$$s$$$ ($$$2 \le |s| \le 2 \cdot 10^{5}$$$). The string consists only of lowercase Latin letters "<span class="tex-font-style-tt">a</span>" and/or "<span class="tex-font-style-tt">b</span>".</p><p>Additional constraints on the input: </p><ul> <li> the sum of string lengths over all test cases does not exceed $$$2 \cdot 10^{5}$$$. </li></ul></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output one of the following:</p><ul> <li> <span class="tex-font-style-tt">YES</span>, if it is possible to obtain an <span class="tex-font-style-it">alternating</span> string from $$$s$$$; </li><li> <span class="tex-font-style-tt">NO</span> otherwise. </li></ul><p>You may output each letter in any case.</p></div><div class="sample-tests"><div class="section-title">Example</div><div
