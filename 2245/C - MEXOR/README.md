Then, $$$f(0) \oplus f(1) \oplus \ldots \oplus f(n-1)$$$ should equal $$$k$$$, where $$$\oplus$$$ denotes the <a href="https://en.wikipedia.org/wiki/Bitwise_operation#XOR">bitwise XOR operation</a>. </li></ul> <p><span class="tex-font-style-bf">Note that</span> a permutation of length $$$n$$$ is an array consisting of $$$n$$$ distinct integers from $$$0$$$ to $$$n-1$$$ in arbitrary order.
