<div class="header"><div class="title">A. Yes or Yes</div><div class="time-limit"><div class="property-title">time limit per test</div>1 second</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><p>Last Christmas, your friend Fernando gifted you a string $$$s$$$ consisting only of the characters $$$\mathtt{Y}$$$ and $$$\mathtt{N}$$$, representing "Yes" and "No", respectively.</p><p>You can repeatedly apply the following operation on $$$s$$$: </p><ul> <li> Choose any two adjacent characters and replace them with their <a href="https://en.wikipedia.org/wiki/Logical_disjunction">logical OR</a>. </li></ul><p>Formally, in each operation, you can choose an index $$$i$$$ ($$$1 \leq i \leq |s|-1$$$), remove the characters $$$s_i$$$ and $$$s_{i+1}$$$, then insert:</p><ul> <li> A single $$$\mathtt{Y}$$$ if at least one of $$$s_i$$$ or $$$s_{i+1}$$$ is $$$\mathtt{Y}$$$; </li><li> A single $$$\mathtt{N}$$$ if both $$$s_i$$$ and $$$s_{i+1}$$$ are $$$\mathtt{N}$$$. </li></ul><p>Note that after each operation, the length of $$$s$$$ decreases by $$$1$$$.</p><p>Unfortunately, Fernando does not want you to combine "Yes OR Yes", as he has experienced trauma relating to a certain song.</p><p>Determine whether it is possible to reduce $$$s$$$ to a single character by repeatedly applying the operation above, <span class="tex-font-style-bf">without</span> ever combining two $$$\mathtt{Y}$$$'s.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 500$$$). The description of the test cases follows. </p><p>The only line of each test case contains the string $$$s$$$ ($$$2\le |s|\le 100$$$). It is guaranteed that $$$s_i = \mathtt{Y}$$$ or $$$\mathtt{N}$$$.</p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, print "<span class="tex-font-style-tt">YES</span>" if the string can be reduced to a single character by repeatedly applying the described operation, and "<span class="tex-font-style-tt">NO</span>" otherwise.</p><p>You can output the answer in any case (upper or lower).
