is, delete the suffix beginning with $$$a_i$$$); </li><li> otherwise, replace $$$a$$$ with $$$[a_{i + 1},a_{i + 2},\ldots,a_m]$$$ (that is, delete the prefix ending with $$$a_i$$$). </li></ul> </li></ul><p>Find the maximum number of coins you can have at the end of the process.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$Here $$$|a_i|$$$ represents the absolute value of $$$a_i$$$: it equals $$$a_i$$$ when $$$a_i  \gt  0$$$ and $$$-a_i$$$ when $$$a_i  \lt  0$$$.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>The first line
