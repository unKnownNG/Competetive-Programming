Kanade's Perfect Multiples</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><div class="epigraph"><div class="epigraph-text"><span class="tex-font-style-it">We've carved those memories into ourselves... No matter how hard they were, they're the lives we carried out!</span></div><div class="epigraph-source">— <span class="tex-font-style-it">Angel Beats!</span></div></div><p>In the afterlife school, Kanade studies a peculiar number game.</p><p>She gives you two integers $$$n$$$ and $$$k$$$, as well as an array $$$a$$$ consisting of $$$n$$$ integers, where $$$1 \le a_i \le k$$$ holds.</p><p>For an integer set $$$B = \{b_1, b_2, \ldots, b_m\}$$$ where $$$1\le b_i\le k$$$, we call it <span class="tex-font-style-it">complete</span> <span class="tex-font-style-bf">if and only if</span> both of the following hold:</p><ul> <li> For each $$$1\le i\le n$$$, at least one divisor of $$$a_i$$$ is contained in $$$B$$$; </li><li> For each $$$1\le j\le m$$$, all positive multiples of $$$b_j$$$ which are less than or equal to $$$k$$$ appear in the array $$$a$$$ at least once. </li></ul><p>You have to find a <span class="tex-font-style-it">complete</span> set $$$B$$$ with <span class="tex-font-style-bf">minimum</span> possible size, or determine that no such set exists.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains two integers $$$n$$$ and $$$k$$$ ($$$1\le n\le 2\cdot 10^5$$$, $$$1\le k\le 10^9$$$) — the length of $$$a$$$ and the upper bound of elements of $$$a$$$.</p><p>The second line contains $$$n$$$ integers $$$a_1, a_2, \ldots, a_n$$$ ($$$1\le a_i\le k$$$) — the elements of $$$a$$$.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2\cdot 10^5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case:</p><ul> <li> If no <span class="tex-font-style-it">complete</span> set $$$B$$$ exists, print a single integer $$$-1$$$ in the only line of output. </li><li> Otherwise: <ul> <li> First print a single integer $$$m$$$ ($$$1\le m\le n$$$) in the first line of output — the size of $$$B$$$. Note that you have to minimize the size of $$$B$$$. </li><li> Then output $$$m$$$ integers $$$b_1, b_2, \ldots, b_m$$$ ($$$1 \le b_i \le k$$$) in the second line — the set you constructed. </li></ul>
