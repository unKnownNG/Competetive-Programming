Niko's Tactical Cards</div><div class="time-limit"><div class="property-title">time limit per test</div>1.5 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Niko is playing a game. Her score is denoted by an integer $$$k$$$ which is $$$0$$$ initially.</p><p>The game has $$$n$$$ turns. On the $$$i$$$-th turn, Niko is given a red card with an integer $$$a_i$$$ on it, as well as a blue card with an integer $$$b_i$$$ on it. She must choose exactly <span class="tex-font-style-bf">one</span> of the cards and update her score according to her choice:</p><ul> <li> If she chooses the red card, her score becomes $$$k - a_i$$$, where $$$k$$$ is her score before the turn. </li><li> If she chooses the blue card, her score becomes $$$b_i - k$$$, where $$$k$$$ is her score before the turn. </li></ul><p>After this, the game proceeds to the next turn, or ends if it is the $$$n$$$-th turn.</p><p>Your task is to find the maximum possible score Niko can obtain at the end of the game.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^3$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$1 \le n \le 10^5$$$) — the number of turns.</p><p>The second line of each test case contains $$$n$$$ integers $$$a_1, a_2, \ldots, a_n$$$ ($$$-10^9 \le a_i \le 10^9$$$).</p><p>The third line of each test case contains $$$n$$$ integers $$$b_1, b_2, \ldots, b_n$$$ ($$$-10^9 \le b_i \le 10^9$$$).</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$10^5$$$.</p></div><div class="output-specification"><div
