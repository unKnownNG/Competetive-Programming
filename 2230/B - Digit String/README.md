For example, the strings <span class="tex-font-style-tt">31</span>, <span class="tex-font-style-tt">222</span>, <span class="tex-font-style-tt">213</span> are beautiful, while the strings <span class="tex-font-style-tt">143</span>, <span class="tex-font-style-tt">3123</span>, <span class="tex-font-style-tt">1322</span> are not. beautiful.</p></div><div class="input-specification"><div class="section-title">Input</div><p>The first line contains a single integer $$$t$$$ ($$$1 \le t \le 10^4$$$) — the number of test
