Game with a Fraction</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>Alice and Bob have two integers $$$p$$$ and $$$q$$$, and they are playing a game with these numbers. The players take turns, with Alice going first. On their turn, a player can do one of two actions:</p><ul> <li> decrease $$$p$$$ by one (this action is possible if $$$p  \gt  0$$$); </li><li> decrease $$$q$$$ by one (this action is possible if $$$q  \gt  1$$$). </li></ul><p>The game ends when $$$p = 0$$$ and $$$q = 1$$$.</p><p>Bob wins if at any point during the game the fraction $$$\frac{p}{q}$$$ is equal to <span class="tex-font-style-bf">in value</span> the fraction $$$\frac{2}{3}$$$. Otherwise, Alice wins.</p><p>Given the initial values of $$$p$$$ and $$$q$$$, determine the winner of the game if both players play optimally.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>Each input case consists of a single line containing two integers $$$p$$$ and $$$q$$$ ($$$1 \le p, q \le 10^{18}$$$).</p></div><div
