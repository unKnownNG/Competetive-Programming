<div class="header"><div class="title">A. Find Minimum Operations</div><div class="time-limit"><div class="property-title">time limit per test</div>1 second</div><div class="memory-limit"><div class="property-title">memory limit per test</div>256 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>You are given two integers $$$n$$$ and $$$k$$$.</p><p>In one operation, you can subtract any power of $$$k$$$ from $$$n$$$. Formally, in one operation, you can replace $$$n$$$ by $$$(n-k^x)$$$ for any non-negative integer $$$x$$$.</p><p>Find the minimum number of operations required to make $$$n$$$ equal to $$$0$$$.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. We can perform the following sequence of operations:</p><ol> <li> Subtract $$$2^0 = 1$$$ from $$$5$$$.
